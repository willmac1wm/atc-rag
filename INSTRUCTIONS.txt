2. Click "Create new" → "Chat prompt"

4. Paste this:

  "description": "Search FAA ATC training materials",
  "parameters": {
    "type": "object",
    "properties": {

        "type": "string",
        "description": "ATC question to search for"

    },
    "required": ["question"]
  }

5. Set endpoint: https://your-url.railway.app/search
